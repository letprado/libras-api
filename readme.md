# Libras API

Este repositório contém a entrega da sprint do projeto Libras API.

O backend principal está na pasta `libras-api/`. Foi nessa aplicação que eu desenvolvi a API REST para cadastro de usuários, controle de sessões de interpretação em Libras, feedbacks, agendamentos e relatórios.

O projeto foi feito com:
- Java 17
- Spring Boot
- Spring Security com JWT
- Spring Data JPA
- Oracle Database
- RabbitMQ
- OpenFeign
- Swagger
