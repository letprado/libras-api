# Libras API

Este repositório contém o projeto da API de gerenciamento de sessões de interpretação em Libras.

O código principal está em `libras-api/`.

Principais entregas do projeto:
- autenticação com JWT;
- sessões e feedbacks;
- agendamento e relatórios;
- integração com RabbitMQ e Feign;
- documentação Swagger em português.
